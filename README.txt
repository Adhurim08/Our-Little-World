OUR LITTLE WORLD — iPHONE GIFT APP

WHAT THIS IS
A private, offline-friendly web app made for Montse and Adhurim. It includes:
- Daily love notes
- “Open when…” letters
- A Costa Rica ↔ Switzerland love map
- A countdown to your next reunion or trip
- A memory gallery with photo uploads
- A love button with heart animation and vibration
- A voice-note upload
- A date-locked secret surprise
- A built-in customization screen

IMPORTANT
The app stores custom text, photos, and audio locally on the device/browser where it is used.
Clearing Safari website data can remove those custom additions.

QUICK TEST ON A COMPUTER
1. Unzip the folder.
2. Open a terminal inside the folder.
3. Run:
   python -m http.server 8000
4. Open http://localhost:8000

PUT IT ONLINE FOR FREE
Because iPhone Home Screen apps need a secure HTTPS website, upload the folder to one of:
- GitHub Pages
- Netlify
- Cloudflare Pages
- Vercel

After deployment, open the HTTPS link in Safari on the iPhone.

ADD TO IPHONE HOME SCREEN
1. Open the deployed link in Safari.
2. Tap the Share icon.
3. Tap “Add to Home Screen”.
4. Tap “Add”.

CUSTOMIZE
Tap the small gear icon in the top-right corner.
You can change:
- Both names
- Main quote
- Next reunion date
- Secret surprise date and message
- Daily love notes

Tap any memory card to edit it.
Tap “+ Add” beside Memory Garden to add photos.
Tap “Add or play a voice note” to upload a recording from iPhone Files.

FILES
- index.html — app structure
- styles.css — visual design
- app.js — app features and sample content
- manifest.json — Home Screen app configuration
- service-worker.js — offline support
- icons/ — app icons
